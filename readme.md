Les fonctionnalités de ma calculatrice :
	- on clique sur un bouton puis sur un opérateur ensuite sur un autre boutton;
N.B: les opérateurs ne s'affichent pas sur la calculatrice;
PS: toutes les fonctions de base marchent en dehors de la division (qui effectue plutot une division entière)


Aucune autre fonction ne marche (tan, sin, arcsin,...) idem pour les paranthèses et les points. 
De même , les touvhes DEL et ON ne sont pas encore opérationelles;

le langage utilisé est java avec la technologie Javafx.
